Myntra Clone 
A comprehensive e-commerce platform inspired by Myntra, built using a robust Java backend and a clean, responsive HTML/CSS frontend.

Tech Stack
Frontend: HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla)

Backend: Java (Spring Boot / Servlets & JSP)

Key Features
1. User Interface (HTML/CSS)
Pixel-Perfect Design: Replicating the iconic Myntra navigation bar and multi-column layouts.

Responsive Grids: Product listings that adapt from 4 columns on desktop to 1 column on mobile.

Interactive Components: CSS-based hover effects on product cards and sliding image banners.

2. Backend Logic (Java)
Session Management: Secure user login and persistent "Add to Cart" functionality using Java Sessions.

Product Management: REST APIs to fetch products by category, gender (Men/Women/Kids), and brand.

Order Processing: Logic to handle price calculations, tax, and discount coupons.
